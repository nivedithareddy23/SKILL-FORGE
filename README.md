# 🚀 SkillForge – AI-Driven Adaptive Learning & Exam Generator

SkillForge is an AI-powered e-learning platform that personalizes learning paths, auto-generates quizzes, and provides real-time analytics.

## Tech Stack
- Frontend: Angular
- Backend: Node.js
- Database: MySQL
- AI: Grok API

## Features
- Adaptive learning engine
- AI-based quiz generation
- Real-time analytics dashboard
- Role-based authentication
